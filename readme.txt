== Description ==

https://www.youtube.com/watch?v=gQhWpZKEIhw

A WooCommerce isotope filter is an Elementor widget designed to enable product filtering using the isotope layout. This feature is ideal for creating a sleek and modern shop or online store. The widget offers various customization options for the filtering experience, such as setting the number of products to display, determining the sorting order, and specifying filtering criteria. Additionally, you can modify the widget's style to suit your design preferences.

For more details :[WooCommerce Isotope Filter for Elementor](https://wpocean.com/wp/plugins/wco-isotope-filter).
For our other elementor & WP works visit : [wpOcean](https://wpocean.com).
